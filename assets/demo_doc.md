# Introduction

**md2star** is an elegant bridge between the flexibility of Markdown and the rigidity of corporate Microsoft Office templates.

## 🌟 Why use md2star?
Writing in Markdown is fast and interruption-free. But the corporate world runs on PDFs and DOCX files.
By using this toolset, you get:
1. **Speed:** Write your thoughts down fast.
2. **Quality:** Automatically generated, high-fidelity corporate documents.
3. **Typography:** Built-in LaTeX math support like this: $\int_{a}^{b} x^2 dx$

### Code Samples
```bash
# This is how simple it is:
md2docx my_report.md
```

Stop wasting time formatting. Start writing.
